Repository Description:
This repository contains the source code and assets for "Thrones Types," a web application that allows users to explore the enneatypes of Game of Thrones characters. Discover the psychological characteristics of your favorite characters from the world of Westeros!

Key Features:
- Explore character enneatypes.
- Select your favorite house to reveal character details.
- Responsive design for various screen sizes.
- Data fetched from the ThronesAPI.
